# overflow.css

CSS `overflow` classes

## classes

- `.overflow-visible`
- `.overflow-hidden`
- `.overflow-scroll`
- `.overflow-auto`

## setup

```
npm install overflow.css
```

```css
@import 'node_modules/overflow.css/overflow';
```

## usage

```html
<div class="overflow-auto">
  example
</div>
```
